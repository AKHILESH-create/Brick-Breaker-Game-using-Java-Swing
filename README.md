# Brick Breaker Game using Java Swing

## 🧱 Overview

The **Brick Breaker Game using Java Swing** is a GUI-based arcade game developed using **Java Swing**, **AWT**, and **Object-Oriented Programming (OOP)** concepts. The game challenges players to break all bricks using a bouncing ball controlled by a movable paddle.

This project focuses on real-time game mechanics, collision detection, keyboard interaction, score tracking, and level-based gameplay implementation.

---

## ✨ Features

* 🎮 Interactive paddle and ball movement
* 🧱 Brick collision and destruction system
* 📈 Real-time score tracking
* 🏆 Multiple game levels
* 💀 Game Over and Restart functionality
* ⌨️ Keyboard controls for paddle movement
* ⚡ Dynamic gameplay speed
* 🖥️ Java Swing based graphical interface

## ⚙️ How the Game Works

1. The player controls the paddle using keyboard keys.
2. The ball bounces around the screen.
3. Bricks break when hit by the ball.
4. The player gains points for each brick destroyed.
5. The game progresses through levels with increasing difficulty.
6. The game ends if the ball touches the bottom boundary.
